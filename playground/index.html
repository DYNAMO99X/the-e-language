What is E?

E is an English-like interpreted language designed for beginners. No semicolons, no braces — just plain English.

say "Hello, world!"
let name be "Ada"
say "Hi, " , name

Playground Limitations

This browser playground runs E via Pyodide (Python in WebAssembly). Some features are restricted:

  • Input — ask does not work (no stdin in browser)
  • Window turtle — only headless/text mode is available
  • Performance — slower than the native Python version
